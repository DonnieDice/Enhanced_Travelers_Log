**ETL | Enhanced Traveler's Log**

------------------------------

This addon adds progress bars to the Traveler's Log and includes a draggable minimap button that opens the journal.

- Retail Midnight support
- Traveler's Log progress bars on scrolling activity rows
- `/etl`, `/etl icon on`, `/etl icon off`, `/etl status`, `/etl help`
